# Retro Picnic Date Quest 🧺❤️

An interactive 2D retro-styled chiptune web game built with HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Retro 8-Bit Theme**: Custom pixel graphics, sound effects, and floating heart animations.
- **Quest System**: Walk around the grid and collect the picnic items: Strawberry 🍓, Croissant 🥐, and Coffee ☕.
- **Synthesized Chiptunes**: Leverages the browser's native **Web Audio API** to generate sound effects (hovers, pickups, victories) programmatically—no external assets required!
- **Interactive Proposal Dialogue**: A custom proposal screen where the "No" button runs away and displays funny messages when hovered or touched.
- **Date & Time Picker**: Allows locking in a date and time for the picnic, rendering it dynamically in a cute victory card.
- **Responsive Layout & Mobile Controls**: Includes an on-screen D-pad for mobile/tablet touchscreen compatibility.

## How to Play
1. Clone the repository.
2. Open `index.html` directly in your browser or run a simple local server (e.g., `python -m http.server 8000`).
3. Have fun!
